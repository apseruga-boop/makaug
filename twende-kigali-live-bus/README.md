# TWENDE - Kigali Live Bus

Standalone mobile launch prototype for a Kigali live bus, Tap&Go balance, journey planner, live route tracker, and stop-alert experience.

Open `index.html` directly in a browser. No build step or server is required.

## What is included

- White/blue mobile UI with color-coded route accents.
- Three simple tabs: Card, Plan, Live.
- Card-first home with balance, recent trips, quick top-up, Tap & Go boarding, and alight flow.
- Planner with from/to, fare, next bus, ride time, and a clean route map.
- Live tracker with next bus, distance, crowding, stop alerts, stops, and a moving bus marker.
- Leaflet/OpenStreetMap interactive maps with live route and stop pins.
- API client and go-live integration plan for production connections.

## Files

- `index.html` - the mobile app prototype.
- `api-client.js` - frontend client scaffold for live routes, cards, payments, and alerts.
- `twende-config.example.js` - production config template for maps, Tap&Go, mobile money, and transit feeds.
- `GO_LIVE_API_PLAN.md` - backend/API implementation plan.
- `LICENSE` - proprietary project license.
